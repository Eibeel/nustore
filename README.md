# Nustore

Nustore te permite
explorar entre sus miles de productos, desde su búsqueda y el navegar entre sus diferentes
categorías, hasta observar cada detalle de sus productos de intererés.

- [Explorar](https://nustore.vercel.app/) 🚀

### Stack:
Nustore fue diseñado en Figma, y desarrollado con frameworks populares al rededor de Javascript.

- [Axios](https://axios-http.com/)
- [Chakra UI](https://chakra-ui.com/)
- [Nuka Carousel](https://github.com/FormidableLabs/nuka-carousel)
- [Firebase](https://firebase.google.com/?hl=es)
- [Figma](https://www.figma.com/)
- [Iconoir](https://iconoir.com/)
- [React](https://es.reactjs.org/)
- [React Hook Form](https://react-hook-form.com/)
- [React Redux](https://react-redux.js.org/)
- [React Router](https://reactrouter.com/en/main)
- [Redux Toolkit](https://redux-toolkit.js.org/)
- REST API Mercado Libre
